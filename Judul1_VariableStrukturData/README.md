Program Sportify : Pemutar Playlist Lagu Sederhana

Sportify merupakan program pemutar playlist musik sederhana yang menerima inputan playlist dari user, menggunakan algoritma struktur data singly linked list
<img width="1782" height="3572" alt="code-snapshot" src="https://github.com/user-attachments/assets/79820c52-31b6-469f-899e-735a9a1eeb06" />
Cara kode berjalan :
1. Mengimport library time untuk mengatur waktu delay setiap lagu
2. membuat class lagu untuk mendefinisikan node singly linked list berdasarkan judul dan penyanyi
3. membuat class terdiri dari definisi tambah lagu, tampilkan_playlist, putar_semua
4. lalu mendeklarasikan my_playlist = PlaylistMusik()
5. Mendeklarasikan fungsi main
6. terdiri dari welcome text
7. lalu menerima inputan judul lagu dan penyanyi dari user
8. menyimpan inputan tersebut
9. memberikan pilihan untuk lanjut memasukan lagu atau sudah selesai
10. jika selesai daftar lagu akan langsung ditampilkan
11. lalu otomatis memutar lagu sesuai urutan
12. jika lagu sudah diputar semua maka akan menampilkan "playlist selesai diputar"
